# Video2Gif Changelog

## [1.0.0] - 2026-03-26

### Added

- Quick Convert command — select a video in Finder, trigger the command, get a GIF
- Custom Convert command — form with quality, max width, FPS, speed, loop count, and trim controls
- Two-pass palette generation for high-quality GIF output
- Estimated file size preview before conversion
- Automatic video file detection from Finder selection or clipboard
- Conversion history stored locally
- Configurable output folder, default quality, and long video threshold via preferences
- Support for MP4, MOV, AVI, MKV, WebM, FLV, WMV, M4V, 3GP, and MPEG formats
